# gen-release-notes

gen-release-notes is a simple utility to generate release-notes for K3s and RKE2.

### Examples

Generate release notes for k3s v1.21.5+k3s1

```sh
gen-release-notes -r k3s -m v1.21.5+k3s1 -p v1.21.4+k3s1
```
